# ARC Raiders - Items & Crafting Tool

**Version 2.0.0**

**Visit the live tool:** [https://werd-sire.github.io/arcItems](https://werd-sire.github.io/arcItems)

A free, web-based crafting calculator, profit analyzer, and item database for ARC Raiders. Plan your crafting, track your gathering, compare weapons, and find the most profitable items to craft.

---

## What is This?

This is an **unofficial fan tool** for ARC Raiders that helps you:

- **Browse & Search** - Find items quickly with advanced filtering by type, rarity, and name
- **Plan Crafting** - View complete crafting trees and calculate all materials needed
- **Track Gathering** - Create shopping lists with optimization suggestions
- **Compare Gear** - Side-by-side weapon stats with automatic DPS calculations
- **Analyze Profit** - Find the most profitable items to craft and sell
- **Track Events** - Live in-game event timers with auto-refresh
- **Plan Loadouts** - Compare augments and shields for your builds

## Features

### Browse Items
Search through 500+ items with detailed information including:
- Stats and attributes
- Crafting recipes and materials
- Where to find items (traders, ARCs, loot areas)
- Weapon specs (ammo type, firing mode, armor penetration)
- Recycle values

### Crafting System
- Complete recipe database for all craftable items
- Visual crafting trees showing all intermediate steps
- Automatic calculation of base materials needed
- Add all materials to shopping list with one click

### Shopping List & Projects
- **Projects**: Track what you're crafting (your goals)
- **Shopping List**: Track materials you need to gather
- **Optimization**: Find recycling shortcuts (e.g., "ARC Coolant recycles to 16x Chemicals")
- **Smart Conversion**: Convert craftable items to their base materials
- Persistent browser storage - your lists never get lost

### Weapon Comparison
- Compare up to 4 weapons side-by-side
- **4 DPS Metrics**: Burst DPS, Effective DPS, Magazine Damage, Time to Empty
- 50+ comparable stats with proper formatting
- Visual stat bars and best/worst highlighting
- Weapon specs from multiple data sources for accuracy

### Profit Analysis
- Analyze 170+ craftable items for profit potential
- **From-scratch costs**: Full material breakdown including upgrade chains
- Sort by profit, ROI, or item type
- Filter by workbench and rarity
- Accurate calculations accounting for tiered weapon upgrades

### Event Timers
- Live event data with auto-refresh
- Filter by map location
- Active/upcoming status indicators
- Countdown timers to next events

### Loadout Planning
- Compare augments by tier and stats
- Compare shields (Light, Medium, Heavy)
- Side-by-side stat comparison tables

## How to Use

1. **Browse Tab**: Search for items, expand cards for details, add to shopping list or comparison
2. **Craft Tab**: Select a recipe to see materials and crafting tree
3. **Gather Tab**: View shopping list, track progress, use optimization suggestions
4. **Compare Tab**: Select weapons to compare stats and DPS
5. **Profit Tab**: Find the most profitable items to craft
6. **Events Tab**: Check live in-game event timers
7. **Loadout Tab**: Compare augments and shields

All data is saved locally in your browser - no account needed!

## Data Sources

Game data is merged from multiple community sources for accuracy:

- **[Metaforge API](https://metaforge.app/arc-raiders)** - Items, quests, traders, events, ARCs
- **[ardb.app API](https://ardb.app)** - Enhanced weapon stats with base values and bonuses separated

Data is updated periodically. Last update information is shown in the app.
