## Features

### Browse Items
Search through 2,500+ items with detailed information including:
- Stats and attributes
- Crafting recipes and materials
- Quest requirements
- Where to find items (traders, ARCs, loot areas)
- What items are used to craft

### Crafting System
- Complete recipe database for all craftable items
- Visual crafting trees showing all intermediate steps
- Automatic calculation of base materials needed
- Add all materials to shopping list with one click

### Shopping List
- Persistent browser storage - your list never gets lost
- Track items found vs needed while gathering
- Quick increment/decrement controls
- Clear completed items or reset entire list

### Item Comparison
- Compare up to 4 weapons, shields, or augments
- Automatic DPS calculations for weapons
- Side-by-side stat comparison
- Visual rarity indicators

## How to Use

1. **Browse Tab**: Search for items, expand cards for details, add to shopping list
2. **Craft Tab**: Select a recipe to see materials and crafting tree
3. **Gather Tab**: View your shopping list, mark items as found
4. **Compare Tab**: Select items from dropdown or browse tab to compare stats

All data is saved locally in your browser - no account needed!

## Data Source

Game data is provided by the [Metaforge API](https://metaforge.app/arc-raiders), a community-driven database for ARC Raiders game information.

Data is updated periodically. Last update information is shown in the app header.
